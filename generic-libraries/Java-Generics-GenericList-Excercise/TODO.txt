Generic Libraries
---------------------------------------------------------------------

Generic List
---------------------------------------------------------------------

Schreiben Sie eine generische Version der Klasse "DoubleList" namens
"GenericListImpl", welche das Interface "GenericList" implementiert. 

Sie können Ihre Implementierung mit der Klasse "GenericListTest"
überprüfen.

Tipp: Number.longValue() returns the value of the specified number as 
	  a long.
